virtual bool CSGNode::clip ( Ray &  ray_,
IntervalSet &  bounds 
) const
pure virtual

clip a ray by a CSGNode.

Paramètres
ray_the ray to clip
boundsray-space sorted bounds of the resulting clipped segments.
Renvoie
true if ray was clipped. bounds parameter is filled by this method

For complex objects, values in bounds represent an ordered set of in - out events. An in event is when the ray enters the geometry, an out is when the ray leaves the geometry. The first event of this array is an in event, followed by an out and so on.
