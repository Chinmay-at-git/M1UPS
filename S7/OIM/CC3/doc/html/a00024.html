bool Plane::clip ( Ray &  ray_,
IntervalSet &  bounds 
) const
inlinevirtual

clip a ray by a plane.

Paramètres
ray_the ray to clip
boundsray sorted bounds of the resulting clipping segment.
Renvoie
true if ray was clipped. bounds parameter is filld by this method

For simple convex objects, there is two values in bounds that represent in and out events. An in event is whe the ray enters the geometry, an out is when the ray leaves the geometry.
