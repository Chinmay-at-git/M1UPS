Description détaillée

The IntervalSet class.

Un ensemble d'intervalle est un tableau de bornes contenant n éléments permettant de définir n/2 intervalles Les fonctions union_of_intervals(), intersection_of_intervals() et difference_of_intervals() doivent être écrites par les étudiants. Voir DEVOIR : Arbres CSG en lancer de rayons pour les précisions.
